##Étapes pour créer un nouveau commit :##

1. Proposer les changements :

   `git add <filename>`

   ou

   `git add *`

2. Valider les changements :

   `git commit -m "Message de validation"`

3. Envoyer les changements :

   `git push -u origin master`
